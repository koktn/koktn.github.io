ふと、GitHub PagesでBlogを作ろうと思い（結果できあがったのがココ）、参考にしたページをまとめます。

# GitHub Pagesでのブログのデプロイ

blogのデプロイ手順については、こちらを参考にしました。

VuePress を使って GitHub Pages でブログを公開する

サイト用リポジトリ名は [アカウント名].github.io にする必要があります。 (作られるページはhttps://アカウント名.github.io)

# VuePressのテーマ

今回テンプレートには、vuepress-theme-meteorlxy を使いました。

インストール方法は

vuepress-theme-meteorlxy

を参考にしました。

また、日本語の対応に関しては以下のサイトが詳しいです。

VuePressで作ったblogに配布されているテーマを設定する

１点、メニューの日本語化については、'en-JP'ではなく、'ja-JP'に修正されているようです。

// 変更前
themeConfig: {
 lang: 'en-US',
}
// 変更後
themeConfig: {
  lang: 'ja-JP',
  // OR
  lang: require('vuepress-theme-meteorlxy/lib/langs/en-JP'),
}
